# street-mobility-transfer
Code, data and models for street-level mobility interaction transfer across cities with minimal local data.


## Data availability
Due to licensing restrictions (CITYDATA.ai), the full training data are not publicly available.  
The processed San Francisco test dataset is provided in the GitHub Release `v1.0.0` for reproducing the reported metrics and case-study figures.  
For access to the full mobility dataset, please contact CITYDATA.ai.
